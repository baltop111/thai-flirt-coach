✨ Ssom 대화 스타일 분석

실제 대화에서 발견된 핵심 특징:

  • 짧고 가볍게 시작 — 부담 없이 접근
  • 상대방 일정·위치를 자연스럽게 확인
  • "5555" (태국식 ㅋㅋ) 로 유머감각 표현
  • น่ารัก(귀엽다), คิดถึง(보고싶다) 적절히 사용
  • 다음 만남 기약으로 기대감 유지

🧲 캘리브레이션 높이는 법

  • 상대방보다 조금 덜 연락하기
  • 답장이 길어도 짧고 재치있게 답하기
  • 만남 제안은 가볍게 — 거절당해도 쿨하게
  • "왜 생각났지?" 질문으로 상상하게 만들기
  • 이모티콘은 포인트로만 사용

🚨 절대 하지 말 것

  • 답장 없을 때 연속 메시지 금지
  • "나 좋아해?" 직접 질문 금지 (초반)
  • 칭찬을 너무 많이 — 가치가 낮아짐
  • 항상 먼저 연락 — 상대가 연락하게 만들 것
